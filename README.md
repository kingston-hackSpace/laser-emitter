<h1>laser emitter</h1>
<p>A laser emitter as the name of the sensor suggest emits laser, the brightness and strength of the can be controlled using any type of microcontroller.</p>

<p>Set up is pretty easy (if you are not sure check it on the diagram below); connect 5VDC to 5V on the Arduino. Ground to the GND and the signal to any of the digital pins on the microcontroller; Some boards are configured so the left pin is the signal pin.</p>

<h2>Safety issue</h2>
<p>Be careful with using a laser, make sure to use protective equipment and never point the laser at your eyes.</p>
